UI Toolkit is an open-source collection of frontend libraries designed to eliminate repetitive boilerplate in UI development.

## What's Included

### @rhinolabs/react-hooks

A collection of reusable React hooks that simplify common patterns and reduce boilerplate code in your React applications.

### @rhinolabs/ui

Pre-built, accessible UI components that you can use directly in your projects. Built with modern best practices and designed for flexibility.

## Why UI Toolkit?

- **Save Time**: Stop writing the same code over and over again
- **Consistency**: Use battle-tested implementations across your projects
- **TypeScript**: Full TypeScript support for better developer experience
- **Lightweight**: Only include what you need
