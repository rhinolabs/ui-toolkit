## React Hooks

<Tabs>
  <TabItem label="npm">
    ```bash
    npm install @rhinolabs/react-hooks
    ```
  </TabItem>
  <TabItem label="pnpm">
    ```bash
    pnpm add @rhinolabs/react-hooks
    ```
  </TabItem>
  <TabItem label="yarn">
    ```bash
    yarn add @rhinolabs/react-hooks
    ```
  </TabItem>
</Tabs>
