# Getting Started

This guide will help you set up and start using the Rhinolabs UI Toolkit in your React application.

## Prerequisites

- Node.js 16.8 or later
- React 18 or later
- TypeScript 4.5 or later (recommended)

## Installation

You can install the libraries individually based on your needs:

```bash
# Using npm
npm install @rhinolabs/ui @rhinolabs/hooks

# Using pnpm
pnpm add @rhinolabs/ui @rhinolabs/hooks

# Using yarn
yarn add @rhinolabs/ui @rhinolabs/hooks
```

## Usage

### Using UI Components

```tsx
import { Button } from "@rhinolabs/ui";

function App() {
  return (
    <Button variant="default">
      Click me
    </Button>
  );
}
```

### Using Hooks

```tsx
import { useToggle } from "@rhinolabs/hooks";

function App() {
  const [isOpen, toggle] = useToggle(false);

  return (
    <div>
      <button onClick={toggle}>
        {isOpen ? "Close" : "Open"}
      </button>
    </div>
  );
}
```

## Troubleshooting

If you encounter any issues during installation or setup:

1. Ensure all peer dependencies are installed correctly
2. Check that your project meets the minimum version requirements
3. Visit our [GitHub repository](https://github.com/rhinolabs/ui-toolkit) for known issues or to report new ones
